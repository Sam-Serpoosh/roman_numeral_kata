*This is a simple roman numeral kata and below is some basic guidelines for these numbers*

Main Numbers in Roman:

I = 1
V = 5
X = 10
L = 50
C = 100
D = 500
M = 1000

add style sample:
110 => 100 + 10 => CX

subtract style sample:
90 => 100 - 10 => XC


4999 => 4000 + 900 + 90 + 9 => MMMM + CM + XC + IX => MMMMCMXCIX
